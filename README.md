# assah.nvim

 Neovim configuration.

## Install

```bash
git clone https://github.com/adorsys123/assah.nvim.git ~/.config/nvim
nvim
```

Requires [ripgrep](https://github.com/BurntSushi/ripgrep). Plugins install automatically on first launch via lazy.nvim.

## Languages

Go, Rust, Zig, TypeScript, Python, Lua — LSP servers managed through Mason.
